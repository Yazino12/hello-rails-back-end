# Hello Rails Backend

An API for a simple message generator. This rails app is created with --api and intended for backend use.

## Built With

- Major languages (Ruby)
- Framworks (Ruby on Rails)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A text editor(preferably Visual Studio Code)

### Install

- Ruby
- Ruby on Rails
- PostgresSQL

### Using it Locally

- Clone the repository using the below commands.

- `https://github.com/Yazino12/hello-rails-back-end.git `
- `cd hello-rails-back-end`
- `rails s`

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
```

### Usage

Start server with:

```
rails server
```

Visit http://localhost:3000/ in your browser.
